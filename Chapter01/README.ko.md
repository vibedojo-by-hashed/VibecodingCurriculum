## AI 코딩이란?

예전에는 코드를 직접 한 줄씩 작성해야 했습니다. 문법을 외우고, 에러를 수정하고, 검색하고... 학습에 수년이 걸렸습니다.

지금은 다릅니다.

**AI에게 "이런 것을 만들어달라"고 말하면, AI가 코드를 작성합니다.**

### 왜 이게 가능할까요?

Claude 같은 LLM(대규모 언어 모델)은 인터넷의 수많은 코드를 학습했습니다. GitHub의 오픈소스, Stack Overflow의 답변, 기술 블로그 등 수십억 줄의 코드를 봤습니다.

그래서 "파란색 제목이 있는 웹페이지"라고 말하면:
- "파란색 제목" → `color: blue` 패턴
- "웹페이지" → HTML 구조 패턴

이렇게 학습한 패턴을 조합해서 코드를 생성합니다. 마법이 아니라 **패턴 매칭**입니다.

물론 아무렇게나 말하면 안 되고, 잘 표현해야 잘 만들어줍니다. 이 커리큘럼이 필요한 이유입니다.

### 예시: 간단한 웹페이지 만들기

**예전 방식:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>내 페이지</title>
    <style>
        body { font-family: sans-serif; }
        h1 { color: blue; }
    </style>
</head>
<body>
    <h1>안녕하세요</h1>
    <p>제 이름은 홍길동입니다.</p>
</body>
</html>
```
이 모든 것을 외워서 직접 작성해야 했습니다.

**AI 코딩 방식:**
```
"파란색 제목으로 '안녕하세요'가 나오고,
그 아래 '제 이름은 홍길동입니다'가 나오는 웹페이지 만들어줘"
```
LLM은 이런 요청을 받으면 학습한 HTML/CSS 패턴 중에서 가장 적합한 것을 조합합니다. 웹페이지 구조(`<html>`, `<body>`), 제목 스타일(`color: blue`), 텍스트 배치 등을 이미 알고 있기 때문입니다.

## Vibecoding이란?

**Vibecoding**은 AI와 대화하면서 코드를 만드는 새로운 방식입니다.

"Vibe"는 느낌, 분위기라는 뜻입니다. 정확한 문법을 몰라도, 원하는 "느낌"을 전달하면 AI가 이해하고 구현한다는 의미입니다.

### Vibecoding의 핵심

1. **의도 전달**: "버튼 클릭하면 숫자가 올라가게 해줘"
2. **대화하며 수정**: "숫자가 너무 작아. 크게 해줘"
3. **협업**: AI가 제안하고, 사용자가 선택하고, 다시 AI가 수정

기존 코딩은 혼자 모든 것을 알아야 했습니다. Vibecoding은 AI라는 파트너와 함께 만듭니다.

## Claude Code란?

**Claude Code**는 Anthropic이 만든 AI 코딩 도구입니다.

### 웹 채팅(ChatGPT, Claude.ai)과의 차이

ChatGPT나 Claude.ai 같은 웹 채팅에서도 코드를 생성할 수 있습니다. 하지만 불편합니다:

| 웹 채팅 | Claude Code |
|---------|-------------|
| 코드를 복사해서 파일에 붙여넣어야 함 | 파일을 직접 생성/수정 |
| 로컬 파일 접근 불가 | 프로젝트 전체를 파악 |
| 명령어 실행을 직접 해야 함 | 명령어를 대신 실행 |
| 매번 상황 설명 필요 | 현재 상황을 인지 |

**요약**: 웹 채팅은 "레시피를 알려주는 사람", Claude Code는 "직접 요리하는 셰프"입니다.

### Claude Code로 할 수 있는 것

- 웹사이트 만들기
- 앱 만들기
- 파일 정리 자동화
- 데이터 처리
- 기존 코드 수정
- 버그 찾아서 수정

## 학습 성과

25개 챕터를 완료하면 다음을 할 수 있습니다:

- **포트폴리오 웹사이트** 제작 및 배포
- **간단한 앱** (TODO 앱, 게임 등) 개발
- **자동화 도구** (파일 정리, 데이터 처리) 구축
- **AI 서비스** (문서 요약기, 번역기) 구현

코딩을 "배우는" 것이 아니라, **AI와 함께 "만드는" 것**을 학습합니다.

## 시작 전 마음가짐

### 1. 완벽하지 않아도 됩니다
처음에는 이상한 결과가 나올 수 있습니다. 다시 말하면 됩니다.

### 2. 에러는 당연합니다
코딩에서 에러는 실패가 아니라 과정입니다. AI가 에러도 수정해줍니다.

### 3. 작게 시작하세요
처음부터 대단한 것을 만들려고 하지 마세요. "Hello World"부터 시작합니다.

### 4. 궁금하면 물어보세요
"이게 뭐야?", "왜 이렇게 해?" AI에게 물어보면 설명해줍니다.
