## ðŸ–¥ï¸ Computer Basics

| Term | Definition | Simple Analogy |
|------|------------|----------------|
| **Terminal** | A text-based window where you type commands to control your computer | Like texting with your computer instead of clicking |
| **CLI** | Command Line Interface. Same as terminal | The "type commands" way |
| **GUI** | Graphical User Interface. Click-based interface | The "click icons" way |
| **Folder/Directory** | A container that holds files | A drawer in a filing cabinet |
| **Path** | The address showing where a file or folder is located | Like a street address (e.g., /Users/you/Documents) |
| **File Extension** | The part after the dot in a filename (.txt, .html, .js) | A label showing the file type |
| **Root** | The topmost folder. `/` on Mac/Linux, `C:\` on Windows | The ground floor of a building |
| **Home Directory** | Your personal user folder | Your bedroom in the house |
| **Environment Variable** | A system-wide setting stored as a name-value pair | A sticky note your computer always remembers |
| **Executable** | A file that can be run as a program | A machine you can turn on |
| **Process** | A running program | A machine that's currently on |
| **Permissions** | Rules about who can read, write, or run a file | Door locks and keys |
| **Shell** | The program that interprets your terminal commands | The translator between you and the computer |
| **Bash/Zsh** | Popular shell programs | Different translators (slight accent differences) |
| **stdin/stdout/stderr** | Standard input, output, and error streams | The keyboard, screen, and error message channels |
| **Pipe (\|)** | Sends output from one command to another | A conveyor belt between machines |
| **Flag/Option** | A modifier that changes how a command works (e.g., `-v`, `--help`) | Settings dial on a machine |
