# Ops IDs (Optional)

> This document is for ops reference only and does not imply any required changes to the curriculum.

운영에서 특정 위치를 참조할 때 쓸 수 있는 naming convention입니다.
**커리큘럼 내용에는 영향 없음. 강제 없음.**

## ID 형식

`ch{NN}-{type}-{slug}`

- `ch01`, `ch02`, ... `ch30`
- type: `section`, `step`
- slug: `kebab-case` (예: `api-key-setup`)

## 예시

- `ch02-section-api-key-setup`
- `ch03-step-mode-switch`

## 메타데이터 파일

현재는 아무 것도 필요하지 않습니다.
